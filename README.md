### 🎨 Task 1: Digital Literacy Infographic

Created an infographic using Canva to explain digital literacy concepts, including safe internet practices, useful tools, and professional communication. This task improved my ability to present information visually.

Developed professional profiles on GitHub, LinkedIn, and Kaggle. This helped in understanding the importance of building an online presence for academic and career growth.
